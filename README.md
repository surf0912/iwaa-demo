# IWAA Calculator — Demo

A single-file demo of the IWAA (Income-Weighted Average Age) calculator.
Everything is in `index.html` — no build step, no dependencies, no server needed.

## View online (GitHub Pages)

1. Create a new **Public** repo on GitHub.
2. **Add file → Upload files** → drag in `index.html` → **Commit changes**.
3. **Settings → Pages** → Source: **Deploy from a branch** → Branch: **main** / **/(root)** → **Save**.
4. Wait ~1 minute. Your link will be:

   ```
   https://<your-username>.github.io/<repo-name>/
   ```

Open it in Safari/Chrome on any phone or computer. After it loads once, it keeps
working on weak or no connection (it's a fully self-contained page).

## Run locally

Just double-click `index.html`, or open it in any browser.
